# JubMate FE

## Installation

Recommend to use [pnpm](https://pnpm.io/) as package manager.

```bash
pnpm install
```

## Development

```bash
pnpm run dev
```

## Build

```bash
pnpm run build
```

## Note when developing

Pages: Add pages in `src/pages` folder. See https://uvr.esm.is/guide/file-based-routing.html for more details.
